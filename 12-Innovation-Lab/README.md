# 12 · Innovation Mini-Lab 🔬

> A safe space to experiment. Goal: make the game **as gripping as chess** and **as elegant as Go** —
> while keeping Kharbga's soul (custodial capture, no luck, North-African heritage).

## The golden rule of this lab
**We do NOT replace the authentic game — we layer on top of it.**
- `Classic Mode` = 100% authentic Kharbga (our cultural-preservation promise; never touch it).
- `Origins Mode` = the enhanced, modern game where these experiments live.

That way we keep the heritage *and* fix "boring." Two modes, one app.

## Honest first principle (read before designing)
You played the **kharbga.com** version and found it boring. Part of that is the *rules* — but a big part is
that it's an **ugly 2005-era web page with a weak AI and no game-feel**. Chess is also "boring" on a bad site
against a dumb bot. So "make it fun" = **rules depth + great AI + juice (animation, sound, feedback) + good UX.**
Don't blame only the rules; ~half the fix is presentation. We attack both.

## How we'll use the lab
1. Brainstorm freely here (cheap).
2. Pick 1–2 experiments → write a tight rule delta.
3. Prototype fast in the TypeScript engine (a variant flag).
4. Playtest → keep only what's genuinely more fun. Kill the rest without mercy.
